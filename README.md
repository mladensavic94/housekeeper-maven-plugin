# housekeeper-maven-plugin

## first run
Add the following snippet to settings.xml
```xml
  <build>
    ...
    <plugins>
        ...
      <plugin>
        <groupId>io.github.mladensavic94</groupId>
        <artifactId>housekeeper-maven-plugin</artifactId>
        <version>1.0-SNAPSHOT</version>
      </plugin>
        ...
    </plugins>
    ...
  </build>
```
